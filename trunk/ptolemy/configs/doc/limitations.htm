Limitations in Version 1.0

Version 1.0 has a number of limitations.

Limitations in the Actor Libraries

  • The actor library is far from complete.
  • The Expression actor operates only on doubles at this time.

Limitations in the User Interface

Version 1.0 of Ptolemy II includes the first graphical user interface for Ptolemy II. This UI still has a number of limitations.
  • No undo mechanism.
  • Weak support for multiports in the graph editor.
  • No mechanism to set the width of a relation.
  • No mechanism to create a port in an atomic actor.
  • The integrated web browser is not a full-featured browser.
  • No snap-to-grid.
  • No icon editor.

Missing Domains

  • Synchronous/reactive.
  • Boolean and integer dataflow.
  • Dynamic dataflow
