Expression Evaluation

The following applet evaluates the expression in the box, which is a function of two ramp signals, slow and fast.


If you were able to run applets, you would have one here.

The block diagram is as shown below, where the expression is a parameter of the Expression block:

The slow signal is a ramp that starts at zero and increments by PI/100 on each firing. The fast signal is a ramp that starts at zero and increments by PI/10 on each firing. Try changing the expression in the box to "cos(slow)*cos(fast)" and clicking on the "Go" button. Other interesting alternatives include:

  • "cos(fast*cos(slow))"
  • "0.2*slow + cos(fast)"
