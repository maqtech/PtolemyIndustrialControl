CT domain demo - Lorenz System

Introduction

The Lorenz System is a famous nonlinear dynamic system that shows the unstable periodic orbit.

The system is given by: 

    dx1/dt = sigma*(x2-x1)
    dx2/dt = (lambda-x3)*x1 -x2
    dx3/dt = x1*x2-b*x3

This applet show the chaotic attractor of the system state projected on to the (x1, x2) plane. The initial conditions of the states are all 1.0. The parameters sigma, lambda and b can be explored.
