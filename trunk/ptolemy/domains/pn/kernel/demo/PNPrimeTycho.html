PN Prime Demo

This demo uses the Process Network Domain in Ptolemy II to find primes by using the sieve of Eratosthenes.

To run the demo, you must be viewing this file with Tycho and have Tcl Blend installed locally. Click on the Tcl code below to build and run the Universe. Note that there will be a short delay while the Tcl Blend backend starts up. 

source [file join $TYCHO java pt domains pn kernel demo PNPrimeTycho.tcl] 
PNPrimeTycho 50

What you should see

  • In the Tycho Console, you should see messages from the run.
  • The first Directed Acyclic Graph (DAG) that is displayed shows the structure of the Entities that make up the Universe before it is run.
  • The Second DAG that is displayed shows the structure of the Entities that make up the Universe after it is run.
  • The nodes of both DAGs are mouse sensitive. If you double click on a node, a window that describes the node will appear.

How it works

PNPrimeTycho.tcl is a Tcl file that is sourced by Tycho. This file starts up Tcl Blend Process via the Tycho JRPC interface.

The Tcl Blend Process sources PNPrimeExample.tcl, which contains Tcl Blend commands that build the universe and run it. The universe uses classes in $TYCHO/java/pt/domains/pn/kernel to implement the Process Network domain.

The Example System

To view a DAG representation of the Ptolemy II kernel document example system, click on the Tcl code below: 
source [file join $TYCHO java pt domains pn kernel demo PNPrimeTycho.tcl] 
ExampleSystemTycho
