Discrete Event Queueing Demonstration


If you were able to run applets, you would have a queueuing demo here.

This applet uses the Ptolemy II DE domain architecture to simulate a queueing system with blocking policy. This particular demo is obtained from the Ptolemy 0.7.1 DE domain demo.

This applet is implemented by QueueApplet.java.

The simulation consists of a Clock Actor, a Ramp Actor, DEFIFOQueue Actor, Server Actor, PassGate Actor, Delay Actor, TestLevel Actor, and a Not Actor connected to a Plot Actor.
