Demo of DE simulation applet

DE Simulation:

The following applet uses the Ptolemy II Discrete Event (DE) domain to simulate a simple arbitrary-bit adder circuit.

The waveforms taken from various points in the circuits is shown.

You can interact with the applet by choosing the stop time of the simulation and choosing the two binary numbers to be added. If the two numbers are not of the same length, then the shorter one will be padded on the right with zeros. Note that this might not be the right thing to do in real-life application. Nevertheless, for the purpose of this demo, it should be fine.


 
 


If you were able to run applets, you would have a waveform demo here.

This applet is implemented by AdderApplet.java.

The simulation consists of a Clock Actor, a XORGate Actor, ANDGate Actor, ORGate Actor, FullAdder Composite Actor, WaveForm Actor and a Register Actor connected to a LogicAnalyzer Actor.
