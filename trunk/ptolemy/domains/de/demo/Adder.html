Logic Circuit

This applet uses the Ptolemy II Discrete Event (DE) domain to simulate a simple arbitrary-bit adder circuit. The plot represents the logic levels at various points in the circuit.


If you were able to run applets, you would have a Logic circuit demo here.

You can interact with the applet by choosing the stop time of the simulation and choosing the two binary numbers to be added. If the two numbers are not of the same length, then the shorter one will be padded on the right with zeros. Note that this might not be the right thing to do in real-life application. However, for the purpose of this demo, it should be fine.

This applet is implemented by AdderApplet.java.

The simulation consists of a Clock Actor, a XORGate Actor, ANDGate Actor, ORGate Actor, FullAdder Composite Actor, WaveForm Actor and a Register Actor connected to a LogicAnalyzer Actor.
