PlotML Applet

This applet demonstrates the subset of PlotML that implements commands supported by the PlotBox class, which controls the axes and labeling. Below is an example of a plot configured by an XML file. There is no data plotted because this applet is demonstrating the base class, PlotBox, which cannot plot data. It only manages the axes.
