Plotting Binary Data

The plot below obtains its data from two binary files given by a URL address. This binary format is used in pxgraph, the X11 plotting tool that is part of the Ptolemy Project. For information about the binary format, see the Pxgraph Java class documentation. For information about endian issues, see the Endian Examples.
