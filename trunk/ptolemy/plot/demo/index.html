Plot Examples

The following pages contain examples of the Plot class, which draws various kinds of two-dimensional plots of data.

  • Black and white plot using unique marks
  • Sinusoids plotted with impulses
  • Bar graph
  • Live, animated plot

The main class implementing the plotter is Plot. It is derived from PlotBox, which provides only the axes and docorations of the plot. This is implemented in a base class so that it can be reused for different kinds of plots. Live (animated) data plots are supported by the PlotLive class. This class is abstract; a derived class must be created to generate the data to plot (or collect it from some other application).
