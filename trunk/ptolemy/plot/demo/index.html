Plot Examples

The following pages contain examples of the Plot class, which draws various kinds of two-dimensional plots of data.

Note that because Java is a relatively immature language, and different browsers and installations support different versions, not all of these demos work properly with all browsers. Also, if you have a CLASSPATH environment set when you started your browser, you may get an error message when accessing some of the demos. We suggest unsetting this environment variable, restarting your browser (and sending email to Sun's Java team to complain about the way the CLASSPATH environment variable works). You may also want to check the Java FAQ.
