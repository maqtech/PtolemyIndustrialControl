zpLibrary
a Java library that includes math classes with the following features:
linear fit data and fractal generators. Calculate fractal dimension of lines. DFT (Discrete Fourier Transform). Chaotical functions: Quadratic, Tent Map, Cubic, cSin, cCos..
zpLibrary uses Ptplot to demonstrate some of the results.

http://www.fisica.ufmg.br/~paulinyi/java_ingles.html
