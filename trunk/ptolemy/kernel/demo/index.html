Ptolemy II Kernel Demonstrations

Some of these demos implement the standard Ptolemy II Clustered Example Graph

Some of the examples can be run by executing make tests.

ExampleSystem

The ExampleSystem class is a Java application that creates the Clustered Example Graph and then prints out information about the Ports. For a Tcl version of the Clustered Example Graph, see the CompositeEntity tests.

To run this application, do: 

setenv CLASSPATH ../../..
java ptolemy.kernel.demo.ExampleSystem

ExampleSystemApplet

The ExampleSystemApplet class is an applet implementation of the Clustered Example Graph. Unfortunately because of bugs in JDK1.1.x this applet requires that Ptolemy II be compiled with JDK1.2, so this applet is likely to fail.

To attempt to run the applet, view ExampleSystem.html.

ExampleSerializable

The ExampleSerializable class writes the Clustered Example Graph to disk and then reads it back in.

To run this example, do the following: 

ptII@ptolemy 136% rm *.data
ptII@ptolemy 137% java ptolemy.kernel.demo.ExampleSerializable write
Wrote to ExampleSerializable.data
ptII@ptolemyn 138% java ptolemy.kernel.demo.ExampleSerializable
OK: Description read in from ExampleSerializable.data is the same as the original

ptII@ptolemy 139%
