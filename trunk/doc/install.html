Ptolemy II Installation

The Ptolemy II distribution includes the compiled Java .class files and the javadoc output, so it is not necessary to 'build' or run make to use Ptolemy II

However, if you would like to rebuild from the Java sources, below is an overview

$PTII/mk/tycho.mk contains the makefile variables that are likely to vary between sites. The JAVAHOME variable should be set to the directory that contains your Java Development Kit (JDK).

You can either edit tycho.mk, or call make and pass a new value of JAVAHOME.

For example, to recompile with JDK 1.2beta4: 

cd $PTII
make clean
make JAVAHOME=/opt/jdk1.2beta4
