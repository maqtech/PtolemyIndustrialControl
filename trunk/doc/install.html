Ptolemy II Installation

Rebuilding

The Ptolemy II distribution includes the compiled Java .class files and the javadoc output, so it is not necessary to 'build' or run make to use Ptolemy II

However, if you would like to rebuild from the Java sources, you should run $PTII/configure, which is a GNU autoconf script that looks for your Java compiler, reads in $PTII/mk/ptII.mk.in and generates $PTII/mk/ptII.mk.
