Ptolemy II Directory Structure

ptII
makefile
Top level makefile
adm/
The admin directory, containing release notes, announcements, and so on. This directory is not shipped with the rest of the distribution.
bin/
Scripts used for building and testing
dist/
The distribution directory. The contents of this directory are automatically generated.
bin
Scripts that start ptolemy and ptplot. These scripts are generated by configure, which reads in bin/ptinvoke.in and generates bin/ptinvoke
collections
Doug Lea's Collections Java package. This package predates the JDK1.2 Collections Framework. Ptolemy II 0.4 no longer uses Doug Lea's Collections package, we are including it as a convenience for anyone who used Ptolemy II 0.3. This directory will be removed in the next release.
com
Commercial Java packages, which we are not shipping.
microstar
Microstar Inc's XML Parser
oroinc
Oroinc's regular expression parser, for use with Jacl
config
Used by GNU Autoconf and the configure script
devel
Code that is under development and not to be released.
diva
The diva directory is not shipped with the release.
The Diva (Dynamic Interactive Visualization software system) is used by some of the demos. The Diva jar file is at $PTII/lib/diva.jar For more information, see the Diva section of the installation guide.
doc
The documentation directory
codeDoc
The Java class documentation produced by running cd $PTII/doc; make
coding
This directory that contains this file
design
Location of the Ptolemy II Design doc
img
Images used by various web pages
tutorial
HTML source code for the Ptolemy II Model Tutorial, discussed in the "Building Models" chapter of the Ptolemy II Design document. The Java source code for the tutorial is in $PTII/ptolemy/tutorial
uml
Universal Modeling Language (UML) diagrams of Ptolemy II. Viewing these diagrams require Visio, a commercial program.
lib
.jar files used by Ptolemy, but not shipped
lib/diva.jar
Diva visualization package - used by some of the demos
lib/ptjacl.jar
Jacl, a 100% Java implementation of a subset of Tcl - used by the test suite.
mk
Makefiles that are included in other makefiles
ptII.mk.in
$PTII/configure reads in mk/ptII.mk.in and generates mk/ptII.mk by substituting installation dependent makefile variables, such as the location of the Java Development Kit
ptno-compile.mk
Used in directories that contain source files that are not compiled.
ptdir.mk
Used in directories that contain no source files, only subdirectories.
ptcommon.mk
Contains rules to build derived files from source files.
ptdist.mk
Contains rules to build tar and zip distributions. $PTII/ptolemy/plot/makefile is an example of a makefile that uses ptdist.mk
ptolemy
Java packages used to implement Ptolemy II
actor - documentation
Functional block code
event - documentation
Event Handling (see also kernel.event)
gui - documentation
Graphical User Interface Elements for Models
lib - documentation
Domain Polymorphic actors
sched - documentation
Scheduler classes
util - documentation
Utility classes
data - documentation
Code used to pass data between blocks
expr - documentation
Expression parser
type - documentation
Type system
domains
Various models of computation
csp - Demonstrations - Java documentation: gui - kernel - lib
Communicating Sequential Processes
ct - Demonstrations - Java documentation: gui - kernel - lib
Continuous Time
dde - Demonstrations - Java documentation: gui - kernel - lib
Distributed Discrete Event
de - Demonstrations - Java documentation: gui - kernel - lib
Discrete Event
fsm - Demonstrations - Java documentation: kernel - lib
Finite State Machine
pn - Demonstrations - Java documentation: gui - kernel - lib
Process Networks
sdf - Demonstrations - Java documentation: gui - kernel - lib
Synchronous Dataflow
graph - Java documentation
Non-hierarchical graph classes
gui - Java documentation
Classes for building Graphical User Interfaces in model applets and applications.
kernel - Java documentation
Ptolemy II Kernel
event - Java documentation
Event Handling
util - Java documentation
Kernel Utilities
math - Java documentation
Math classes for Complex numbers and Signal Processing
media - Java documentation
Classes for representing Audio and Image data
moml - Java documentation
Classes for Model Markup Language (MoML) which is used to describe Ptolemy II models.
plot - Java documentation
Ptplot 2-D graphing package
tutorial -
Ptolemy II Tutorial Java code. The html that uses the code is in $PTII/doc/tutorial
util/
Ptolemy II Non-Java Utilities
testsuite
Test suite infrastructure that uses Jacl
vendors
Code from Vendors for commercial products. Not currently shipped.
