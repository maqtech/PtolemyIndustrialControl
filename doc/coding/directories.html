Ptolemy II Directory Structure

ptII
makefile
Top level makefile
adm/
The admin directory, containing release notes, announcements, and so on. This directory is not shipped with the rest of the distribution.
bin/
Scripts used for building and testing
dist/
The distribution directory. The contents of this directory are automatically generated.
bin
Start up scripts for utilities like ptplot
collections
Doug Lea's Collections Java package. This directory will go away once we no longer support JDK1.1
com
Commercial Java packages
microstar
Microstar Inc's XML Parser
config
Used by GNU Autoconf and the configure script
devel/
Code that is under development and not to be released.
diva
The Diva Dynamic Interactive Visualization software system
mk
Makefiles that are included in other makefiles
ptII.mk.in
$PTII/configure reads in mk/ptII.mk.in and generates mk/ptII.mk by substituting installation dependent makefile variables, such as the location of the Java Development Kit
ptno-compile.mk
Used in directories that contain source files that are not compiled.
ptdir.mk
Used in directories that contain no source files, only subdirectories.
ptcommon.mk
Contains rules to build derived files from source files.
ptdist.mk
Contains rules to build tar and zip distributions. $PTII/ptolemy/plot/makefile is an example of a makefile that uses ptdist.mk
ptolemy/
Java packages used to implement Ptolemy II
actor - documentation
Functional block code
util - documentation
Utility classes
data - documentation
Code used to pass data between blocks
expr - documentation
Expression parser
domains
Various models of computation
csp - kernel documentation
Communicating Sequential Processes
ct - kernel documentation
Continuous Time
de - kernel documentation
Discrete Event
dfm - kernel documentation
Design Flow Management
od - kernel documentation
Ordered Dataflow
pn - kernel documentation
Process Networks
sc - kernel documentation
Star Charts
sdf - kernel documentation
Synchronous Dataflow
domains/generic
the icon libraries and demo schematics
graph - documentation
Non-hierarchical graph classes
kernel - documentation
Ptolemy II Kernel
util - documentation
Kernel Utilities
math - documentation
Math classes for Complex numbers and Signal Processing
plot - documentation
Ptplot 2-D graphing package
util/
Ptolemy II Non-Java Utilities
testsuite
Test suite infrastructure that uses Tcl Blend
vendors
Code from Vendors for commercial products. Not currently shipped.
