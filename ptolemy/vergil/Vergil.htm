Vergil Applet

The following applet illustrates how the model visualization used in Vergil can be added to an applet. Here the modulation demo can be run, and the parameters of specific actors can be edited as well.

Note that this applet does not have the full set of commands that are available in the Vergil application. For example, it is not possible to browse the hierarchy of the model. We anticipate enriching this so that the full Vergil application (given Applet security constraints) is accessible from within an Applet

See the MoML file specifying this model, and the MoML file defining a class used in this model.
