The Metro II Ptolemy domain code that uses Google ProtoBuf.

The Google site for Protocol Buffers states:

"Protocol Buffers are a way of encoding structured data in an efficient yet extensible format."

ProtoBuf is only needed if ptolemy/domains/metroII/kernel/util/ProtoBuf/event.proto and ptolemy/domains/metroII/kernel/util/ProtoBuf/metroIIcomm.java needs to be updated.
