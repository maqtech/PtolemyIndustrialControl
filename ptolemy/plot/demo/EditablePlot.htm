Editable Plot

Here is an example of an editable plot:

If your browser recognized the applet tag, you would see an applet here.

At the bottom, you can select one of the three data sets. Then, by dragging the right mouse button, you can change the values plotted for the corresponding data set.

This demonstration is implementing using the EditablePlotMLApplet class in the ptolemy.plot.plotml package. That applet has an applet parameter, dataurl, that specifies the data to be plotted and format information. It creates the choice menu at the bottom and implements the plot using the EditablePlot class instead of the base class Plot.
